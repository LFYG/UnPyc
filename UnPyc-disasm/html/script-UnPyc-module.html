Script UnPyc

UnPyc - program for disassembling and decompiling *.pyc files.

`./UnPyc cVh`:

--== Version ==--
unpyc v0.18 (testing)

--== Help ==--
Usage: ./UnPyc [dvvDgcVh] <file>
d  - disassemble
x  - show xref`s (used with d)
v  - verbose (used with d)
vv - very verbose (used with d)
D  - decompile (not implemented yet)
g  - gui (control flows)
c  - copyright
V  - version
h  - help
