STORE(self, lvalue, rvalue, curIndent, emptystack)

source code 

This method handles all STORE_* commands.

Parameters:
  • lvalue - destination, where data is stored.
  • rvalue - source of data.
  • curIndent - current indent in the generated source code.
  • emptystack - should be True if no postponed stores are expected.

codeDecompile(self, offset=0, length=0, startIndent=0, showStack=False, stack=[], mode='plain')

source code 

Decompiles current code object.

Parameters:
  • offset - start offset in co_code.
  • length - length of the co_code substring to decompile.
  • startIndent - initial indent for the generated source code.
  • showStack - report current stack state after each command.
  • stack - initial stack state.
  • mode - decompilation mode ('plain' or 'conditional').
