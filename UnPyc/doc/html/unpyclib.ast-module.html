Module ast

source code

This module provides classes to build ASTs on decompilation stack.

Classes are divided into two categories: Operator and Data. Operational ones incorporate priority of the operator.

From http://docs.python.org/reference/expressions.html#summary (From lowest to highest priority)

  1. Lambda expression: lambda
  2. Boolean OR: or
  3. Boolean AND: and
  4. Boolean NOT (unary): not X
  5. Comparisons, including membership tests and identity tests, ... : in, not in, is, is not, <, <=, >, >=, <>, !=, ==
  6. Bitwise OR: |
  7. Bitwise XOR: ^
  8. Bitwise AND: &
  9. Shifts: <<, >>
  10. Addition and subtraction: +, -
  11. Multiplication, division, remainder: *, /, %
  12. Positive, negative, bitwise NOT (unary): +x, -x, ~x
  13. Exponentiation: **
  14. Subscription, slicing, call, attribute reference: x[index], x[index:index], x(arguments...), x.attribute
  15. Binding or tuple display, list display, dictionary display, string conversion: (expressions...), [expressions...], {key:datum...}, `expressions...`
