intervals(self)

source code 

Returns list of intervals.

Given a node h, an interval I(h) is the maximal, single-entry subgraph in which h is the only entry node and in which all closed paths contain h. The unique interval node h is called the interval head or simply the header node. (from "Reverse Compilation Techniques" by Cristina Cifuentes).

Currently this method is unused in unpyc project.

Returns:
list of intervals of current CFG.
