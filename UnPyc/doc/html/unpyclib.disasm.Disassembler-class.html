optimizeJumps(self)

source code 

Optimizes conditional jumps.

Replaces JUMP_IF_* -> JUMP_IF_* -> ... -> TARGET with JUMP_IF_* -> TARGET.

optimizeAbsoluteJumps(self)

source code 

Replaces JUMP_ABSOLUTE with CONTINUE_LOOP or NOP.

If JUMP_ABSOLUTE is the last one in the byte code for the specified target, then replace it with NOP, otherwise replace with CONTINUE_LOOP.
